# Button

### Button的类型

```jsx
render() {
    return (
        <Button>primary</Button>
    )
}
```

### Button的尺寸
```jsx
render() {
    return (
       <React.Fragment>
            <Button size='small'>small</Button>
            <Button size='normal'>normal</Button>
            <Button size='large'>large</Button>
       </React.Fragment>
    )
}
```
